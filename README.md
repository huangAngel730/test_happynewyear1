# test_happynewyear1

简洁的静态网页项目，用于展示新年祝福页面。

## 文件说明

- `1`：项目内的资源/占位文件。
- `index.html`：项目主页面，浏览器打开即可访问。
- `script.js`：页面交互逻辑。
- `style.css`：页面样式。
- `LICENSE`：项目许可文件。

## 本地运行

该项目为纯静态页面，直接在浏览器中打开 `index.html` 即可。也可以使用简单的本地服务器：

```bash
python -m http.server 8000
# 然后在浏览器打开 http://localhost:8000
```

或使用 Node.js 的 http-server：

```bash
npx http-server .
```
